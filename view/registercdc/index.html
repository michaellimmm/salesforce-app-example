Step 5: Select Objects for Change Notifications

Now that your foundation is set, let's tailor your integration by choosing the Salesforce objects for Change Notifications. Follow these steps to locate the Change Data Capture (CDC) menu:

Navigate to Change Data Capture:

  1. Access Your Salesforce Account:
    • Log in to your Salesforce account.
  2. Navigate to Setup:
    • In the upper-right corner, click on your profile picture.
    • Select "Setup" from the dropdown menu.
  3. Find Object Manager:
    • In the left sidebar, under "Platform Tools," locate and click on "Object Manager."
  4. Explore Change Data Capture:
    • From the Object Manager, select the Salesforce object you want to subscribe to for Change Notifications.
  5. Check for Change Data Capture Menu:
    • Within the selected object's detail page, look for the "Change Data Capture" menu.

Enable Change Data Capture:

  1. Open Change Data Capture:
    • Once you're in the CDC menu, you'll find a list of fields associated with the selected object.
  2. Select Fields for Change Notifications:
    • Choose the fields you want to monitor for changes and receive real-time notifications.
  3. Save Your Configuration:
    • Don't forget to save your configuration to activate Change Data Capture for the selected object.

Step 6: Register Notification Object

The final touch! Register the selected notification object to ensure you receive relevant Salesforce events seamlessly.

Hold down the Ctrl (windows) or Command (Mac) button to select multiple options.
